HTML Tutorial

Introduction

HTML stands forHypertextMarkupLanguage, and it is the most widely used language to write Web Pages.

  • Hypertextrefers to the way in which Web pages (HTML documents) are linked together. Thus,the link available on a webpage is called Hypertext.

  • As its name suggests, HTML is aMarkup Languagewhich means you use HTML to simply "mark-up" a text document with tags that tell a Web browser how to structure it to display.


    • Originally, HTML was developed with the intent of defining the structure of documents like headings, paragraphs, lists, and so forth to facilitate the sharing of scientific information between researchers.

    Now, HTML is being widely used to format web pages with the help of different tags available in HTML language.





    Let us take the example:


    Example:

    <HTML >

    <Head >
    <title> Example </title>
    </head>

    <body>

    <H1> This is heading </H1>
    <p>This is body </p>

    </body>
    </HTML>



    In the above example:

  • <HTML>: This tag is used as the root element of an HTML page.
  • <Head >: This contains meta information about the HTML page.
  • <title>: This tag specifies a title for the HTML page.
  • <body>: This tag defines the document's body, and its all the visible contents, such as headings, images, hyperlinks, tables, lists, etc.
  • <H1>: This tag used to define large heading.
  • <p>: This tag is used to define paragraphs.

    • What is the Element?


       HTML Element is written in between starting Tag, some content, and an ending tag.
    Syntax:
    <TAG NAME> Element is written here </TAG NAME>
    Example:
    <H1> This is heading </H1>

    Note: some HTML element has no content like <BR>. These tags are called Empty Tags.
